vue支持声明式渲染，采用的是简洁模板语法，并且是响应式的

响应式:当数据发生变化的时候，页面会自动随之变化

普通文本展示

{{message}}

展示html:v-html指令

{{html}}

属性绑定使用v-bind指令

v-bind可以简写为:

数据绑定支持js表达式

{{age+1}}

{{age>18?'成年':'未成年'}}

{{name.split('').reverse().join('')}}

属性绑定也支持js表达式

{{Date.now()}}

指令可以添加修饰符，也就是后缀；例如.prevent可以阻止默认行为

计算属性:html视图模板中不建议写复杂逻辑，当需要进行逻辑运算时，可以使用计算属性
